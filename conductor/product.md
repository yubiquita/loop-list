# Initial Concept

繰り返し使用するチェックリスト管理アプリ

## 1. Vision & Core Concept
`loop-list` は、家事や日常のルーティン、パッキングリストなど、**「繰り返し使用する手順」**をストレスなく管理するためのモバイルファーストなチェックリストアプリです。

## 2. Target Audience
- **個人ユーザー (ルーティン管理):** 毎日、毎週の習慣や家事を効率化したい人。
- **モバイルユーザー:** 外出先や移動中に、片手で素早く項目をチェックまたは追加したい人。

## 3. Core Values & Goals
- **繰り返しの容易さ:** チェックを一括解除して、何度でも同じリストを再利用できる。
- **シンプルさとプライバシー:** サーバー連携を不要とし、すべてのデータをブラウザの LocalStorage に保存する。
- **モバイルでの入力性:** 物理的なキーボードがない環境でも、スムーズに項目を追加・編集できる。

## 4. Key Features (Current & Planned)
- **Current:**
  - リストの作成・編集・削除
  - 項目の追加・チェック・進捗表示
  - **タスクのグループ化:** スワイプによるインデントで、親タスクと子タスクの階層構造を作成。
  - **カスケードチェック:** 親タスクのチェック状態を子タスクに自動連動。
  - LocalStorage によるデータ永続化
- **Planned:**
  - **カテゴリ・フォルダ分け:** 大量のリストを整理・分類。
  - **インポート/エクスポート:** データのバックアップや、別の端末への移行を容易にする。

## 5. User Experience (UX) Strategy
- **モバイルファースト:** タップしやすいボタンサイズ、レスポンシブなレイアウト。
- **ミニマルデザイン:** 迷いを与えない、タスク完了に集中できるインターフェース。
