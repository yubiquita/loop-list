# Specification: Swipe to Delete Feature

## Overview
アイテム（タスク）の削除方法を、従来のボタンクリックから、ジェスチャー（左スワイプ）による操作へと変更し、UIを簡素化すると同時に誤操作を防止します。

## Functional Requirements
1. **ジェスチャーによる削除:**
   - インデントされていない（Level 0/メイン）状態のアイテムを、左方向に大きくスワイプすることで削除を実行する。
   - 削除が実行される閾値は、インデント解除（アウトデント）の閾値よりも深く設定し、意図的な操作であることを保証する。
2. **既存ボタンの撤去:**
   - アイテム上に表示されていた削除ボタン（ゴミ箱アイコン等）を完全に削除する。
3. **視覚的フィードバック:**
   - スワイプ中に、削除が実行されることを示す視覚的な手がかり（背景色の変化、削除アイコンの表示、または「削除」というテキストの表示）を提供する。

## Non-Functional Requirements
- **Mobile Friendly:** タッチ操作によるスムーズなスワイプ体験を提供する。
- **KISS原則の遵守:** 複雑なライブラリを使用せず、既存の Pointer API を活用して実装する。

## Acceptance Criteria
- [ ] インデントされた（Level 1）アイテムを左スワイプした時は、これまで通り「インデント解除」が行われる。
- [ ] インデントされていない（Level 0）アイテムを左スワイプした時に、削除が実行される。
- [ ] 削除ボタンがUIから消えている。
- [ ] スワイプ中に「削除」であることが視覚的にわかる。

## Out of Scope
- 削除の取り消し（Undo）機能の導入（将来的な検討事項とする）。
- ドラッグ＆ドロップによる並び替えへの影響。
